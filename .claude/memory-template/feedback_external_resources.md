name: 외부 리소스 URL 규칙
description: 외부 리소스 URL은 검증된 것만 적고, 모르면 "검색 필요"로 남긴다
type: feedback
---

외부 리소스(Freesound 등) URL·ID·크리에이터명은 다음 세 가지 경우만 허용:

1. **검증된 것만 표기** — 실제 다운로드된 파일이 있거나, 사용자가 직접 확인한 것
2. **모르면 "검색 필요"** — URL 추측·조합 금지. 학습 데이터 기반이라도 확인 불가능한 URL은 적지 않는다
3. **생성 가능한 건 생성** — PowerShell/CLI로 만들 수 있으면 외부 의존 대신 직접 생성

**Why:** Freesound URL을 크리에이터명+ID 패턴으로 추측 조합했다가 전부 깨진 채 사용자에게 전달됨. 할루시네이션. "더 열심히 하려다 할루시네이션이 늘었다"는 피드백.

**How to apply:** 오디오·폰트·라이브러리 등 외부 리소스 URL을 문서에 적을 때마다 "이게 직접 확인된 URL인가?" 체크. 아니면 URL 필드 자체를 생략하고 검색어만 남긴다.
